# Farmsafe

Plataforma operacional rural com:
- rastreabilidade GPS
- abastecimento de cochos
- operação offline
- auditoria operacional
- dashboard web
- aplicativo de campo

## Tecnologias
- React
- TypeScript
- Vite
- Tailwind
- Supabase
- Vercel

## Módulos
- Web Dashboard
- Mobile Collector
- Commercial Landing Page
